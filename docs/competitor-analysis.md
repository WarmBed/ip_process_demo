3. **充實のサポート** — IT + 智財專業支援團隊

#### 對 IP Winner 的啟示

1. **期限自動化是核心** — root ip 的期限設定系統極其精細（標準觸發、國別年金、OA 觸發、馬德里觸發）。IP Winner 若要進入市場，期限自動化至少需達到 TIPO + USPTO 級別
2. **計費系統是黏性來源** — 請求管理（料金表→請求書→入金→送金）是事務所日常運作核心，也是客戶不輕易切換的原因
3. **工作流程引擎是差異** — root ip 有完整的多步驟審批流程，IP Winner 目前缺乏
4. **家族管理是進階需求** — 專利家族（PCT → 各國移行）的管理是國際事務所的基本需求
5. **AI 是 IP Winner 的最大差異化** — root ip 的 AI 僅限操作輔助，IP Winner 的 AI 信件分類 + OA 建議 + 回饋學習是真正的功能差異
6. **定價參考** — root ip 月額 1.65 萬円（~NT$3,500）起，IP Winner 可參考此定價區間

## 八、IP Winner 差異化優勢

### 確認的市場缺口
1. **台灣零競爭** — 全球搜尋無任何台灣本地專利事務所 SaaS
2. **TIPO 原生支援** — 所有競爭者以 USPTO/EPO/JPO 為中心
3. **收發碼自動辨識** — 無競品支援台灣智財局收發文流程

### 功能差異化
4. **AI 信件收發碼分類** — 僅 Anaqua 接近，但不支援 TIPO
5. **AI 建議 + 人工回饋學習** — 無競品具備此功能
6. **Google Drive / Dropbox 原生整合** — 無競品以此為主軸
7. **繁體中文 + 台灣法規期限** — 完全空白

### 定價差異化
8. **SMB 友好** — 主流產品年費 $50K–$100K+ USD，台灣中小型事務所（5–30 人）完全負擔不起

### 最接近的三個參考對象
| 產品 | 參考面向 |
|------|---------|
| **汉知宝** | 功能最接近：代理機構側案件管理、期限追蹤、任務分派 |
| **root ip Cloud** | 定位最接近：雲端 + 專利師開發 + 事務所專用 |
| **超凡 CHOFN** | 概念最接近：完整整合系統（專利+商標+CRM+財務），但自用不外賣 |

## 九、建議策略

1. **MVP 聚焦**：期限預警 + AI 信件分類 + 收發碼管理（這三個是台灣事務所最痛的點）
2. **定價**：對標 AppColl（$100–200/用戶/月），遠低於 Anaqua/Questel
